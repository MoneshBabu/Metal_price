# MetalPrice Tracker

A live precious metal price tracking application built with React, Tailwind CSS, and Framer Motion.

## Features

- **Live Market Data**: Real-time price tracking for Gold, Silver, Platinum, and Palladium.
- **Independent Loading**: Each metal's data is fetched independently with its own loading state.
- **Detailed Analytics**: View previous open/close prices, market dates, and insights.
- **Error Resilience**: Built-in error handling with retry capabilities for each individual metal.
- **Modern UI**: Clean, mobile-first design with smooth animations and high readability.

## Tech Stack

- **React**: UI Framework
- **Tailwind CSS**: Styling
- **Framer Motion**: Animations
- **Lucide React**: Icons
- **React Router**: Navigation

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
